# 卡拉 OK 歌词生成器

因为想做卡拉 OK 视频，同时想试试 AI 编程的流程，所以做了这个软件。

目前仅支持鹅家或类似格式的逐字歌词格式，例如：

> [00:11.869]涂[00:12.077]上[00:12.269]完[00:12.460]美[00:12.768]桃[00:13.177]红[00:13.370]色[00:13.593]的[00:13.785]一[00:14.113]等[00:14.505]星[00:14.754]

我只测试过中文、英文、日文歌，代码也只对中英混合做了适配。

### 已实现的功能

- 歌词淡入淡出
    
- 较长间奏后第一句歌词开始前倒计时
    
- 自动切割较长歌词
    
- 伴唱部分歌词置顶显示
    

### 暂时不支持的功能

- 双语字幕
    
- 带发音的字幕

# 字幕生成方法

1. 首先下载仓库内的 `kolyric.py`，放在一个文件夹内。
    
2. 使用任意 IDE 打开这个文件夹，或直接用记事本打开这个 `.py` 文件进行修改。
    
3. 下载歌词文件，放在 `kolyric.py` 同级文件夹内。
    
4. 按照你想要的需求更改文档头部的设置部分，我已经用注释标明每个部分的设置名称。
    
5. 使用 IDE 或在终端中使用 Python 运行这个 `.py` 程序，运行后即可在 `kolyric.py` 同级文件夹内找到输出好的 `.ass` 文件。

# 字幕使用方法

- **外挂字幕**：直接把字幕名称（`.ass` 前的部分）重命名，保持与歌曲 MV 同名即可。
    

> **注意**：有部分歌曲 MV 会在歌曲的任意部分穿插剧情或对歌曲进行重新编写，会导致歌曲和字幕匹配不上。

## FFmpeg 视频合成指南

我自己的工作流是用指令3。

以下指令均需在终端或命令行中运行。请确保已安装 **FFmpeg**，并根据实际情况替换文件名和参数。

> **注意**：有部分歌曲的伴奏和人声版会出现不重合的选项，可以使用剪辑软件查看这两个音频的波形图并对齐。

### 指令 1: 生成绿幕字幕视频

如果需要将字幕作为独立素材导入剪辑软件，请使用此指令生成带安全截断的绿幕视频：

```Bash
ffmpeg -f lavfi -i color=size=1920x1080:rate=60:color=green -t [这里填入程序输出的歌词时长] -vf "ass=output.ass" -c:v libx264 -pix_fmt yuv420p green_screen.mp4
```

- **可修改项**:
    
    - `size=1920x1080`: 视频分辨率，需与你生成的字幕分辨率保持一致。
        
    - `rate=60`: 视频帧率，若不需要高帧率可改为 30。
        
    - `color=green`: 背景色，若遇到绿色字幕可改为 blue 等。
        

### 指令 2: 合成 MV + 70% 伴奏 + 30% 人声 + 字幕 (静音原视频)

如果你想一步到位输出最终成片，此指令会自动抹除原 MV 的声音，并将你准备好的伴奏与人声按比例混合，最后压制字幕：

```Bash
ffmpeg -i input_mv.mp4 -i instrumental.mp3 -i vocal.mp3 -filter_complex "[1:a]volume=0.7[inst];[2:a]volume=0.3[voc];[inst][voc]amix=inputs=2:duration=longest[aout]" -map 0:v:0 -map "[aout]" -vf "ass=output.ass" -c:v libx264 -c:a aac -shortest final_mixed_mv.mp4
```

- **可修改项**:
    
    - `input_mv.mp4`: 你的原版视频文件名。
        
    - `instrumental.mp3`: 你的伴奏音频文件名。
        
    - `vocal.mp3`: 你提取出的人声音频文件名。
        
    - `[1:a]volume=0.7`: 伴奏音量比例 (0.7 代表 70%)。下一个是人声。
        

### ⭕(推荐)指令 3: 生成特效软件专用底板 (静音原视频 + 仅压制字幕)

导出视频后用专门的视频软件处理声音对齐问题。


```Bash
ffmpeg -i input_mv.mp4 -vf "ass=output.ass" -c:v libx264 -an final_vfx_mv.mp4
```

- **可修改项**:
    
    - `input_mv.mp4`: 你的原版视频文件名。
        
    - `-an`: 此参数为强制静音。
